#  Проект по автоматизации тестирования [halykmarket.kz/](https://halykmarket.kz/)
> Halyk Market — мультикатегорийная онлайн-площадка в Казахстане, 
> созданная банком Halyk Bank. 
> Платформа объединяет покупателей и продавцов, предлагает широкий ассортимент товаров,
> гибкие финансовые условия (рассрочка до 24 месяцев, кредит до 6 млн тенге) и доставку по всей стране
