#  Проект по автоматизации тестирования [Kaspi.kz](https://kaspi.kz/)
> **Kaspi.kz** — ведущая казахстанская экосистема, объединяющая цифровые сервисы
> для платежей, покупок и финансов. Платформа включает маркетплейс, мобильный банк,
> платежные решения и широкий набор онлайн-сервисов, которыми ежедневно пользуются
> миллионы клиентов.
